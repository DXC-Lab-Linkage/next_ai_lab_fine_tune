# NEXT AI LAB
next ai labの取り組みで個人情報をマスクするモデルを作成する。

## 【背景】 
エンタープライズ向けの生成AIアプリケーションを導入する際に、会社で管理している個人情報をLLMにリクエストすることはリスクがある。  
なので、個人情報を直接LLMリクエストせずにマスキングしてLLMにリクエストしているケースがある。  
個人情報をマスキングする工数が多大であり、既存マスキングツールの品質も満足できないレベルである。  
ルールベースでマスキングするともれや、誤ったワードにマスキングしたりする  
マスキングのタスクに特化したLLMを開発し、マスキングの工数削減や品質向上を行いたい。
